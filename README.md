hyperPatterns
=============

Node / express basic app to demonstrate and interact with published laboratory patterns.

####See http://labpatterns.org

Requires node.js v0.12.2, couchdb 1.6.1, 4-store v1.1.5

expects a couchdb instance with the contents of couchdb.zip running on localhost:5984, and 4store running a kb called `patterns` with a sparql http endpoint at localhost:8000/sparql
