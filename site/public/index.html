Laboratory patterns

A framework to communicate and share expert knowledge about lab procedures

Hi there. Write the background and instructions here.

Use the buttons above to navigate between patterns. You can view exisiting patterns or create, edit and publish a new laboratory pattern to share with the world. Note that when editing existing proto-patterns, any associated file attachments will remain unchanged unless you specify a new file to replace them. Click here for a short screencast introducing the basic ideas and function of this site.

Once published, patterns can no longer be changed and are made available as linked data under the URI scheme described here.
